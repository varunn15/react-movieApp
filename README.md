# 🎬 Movie Explorer App

A modern and responsive movie discovery web app built with React.  
Explore movies, watch trailers, and manage your personal favorites with authentication.

---

## 🚀 Live Demo  
👉 (https://moviee-react.vercel.app/)

---

## ✨ Features  

- 🔍 Real-time movie search  
- 🎞️ Browse Popular, Top Rated, Upcoming  
- 🎬 Watch trailers inside the app  
- 🔐 User authentication (Firebase)  
- ❤️ Personalized favorites  
- 🌙 Dark mode toggle  
- 📱 Fully responsive design  
- ⚡ Load more pagination  
- ⌨️ Keyboard shortcuts  
- 🚫 Error handling & loading states  

---

## 🛠️ Tech Stack  

- React  
- Firebase Authentication  
- TMDB API  
- CSS  
- Vercel (Deployment)
