🎬 Movie Explorer App

A modern, responsive movie discovery web app built using React that allows users to explore movies, watch trailers, and manage personalized favorites with authentication.

🚀 Live Demo

👉 Try it here:
https://moviee-react.vercel.app/

✨ Features
🔍 Real-time Search – Instantly search for movies using TMDB API
🎞️ Movie Categories – Browse Popular, Top Rated, and Upcoming movies
🎬 Trailer Integration – Watch official trailers directly inside the app
🔐 Authentication – User login & signup using Firebase
❤️ Personalized Favorites – Save favorite movies per user
🌙 Dark Mode – Toggle between light and dark themes
📱 Fully Responsive – Works across mobile, tablet, and desktop
⚡ Pagination / Load More – Efficient movie browsing
🎯 Clean UI/UX – Smooth and intuitive user experience
⌨️ Keyboard Shortcuts – Faster navigation
🚫 Error Handling & Loading States – Better user feedback
🛠️ Tech Stack
Frontend: React, CSS
Authentication: Firebase Auth
API: TMDB (The Movie Database)
State Management: React Hooks / Context API
Deployment: Vercel

⚙️ Environment Variables

Create a .env file and add:

VITE_TMDB_API_KEY=your_api_key
VITE_FIREBASE_API_KEY=your_key
🧪 Run Locally
git clone https://github.com/your-username/movie-app.git
cd movie-app
npm install
npm run dev
